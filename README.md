# code_for_oscar
Code for OSCAR

The matlab code is a example for this paper <Simultaneous Regression Shrinkage, Variable Selection,
and Supervised Clustering of Predictors with OSCAR>. I modified it so python can use it. If you want to run this code, this linking is helpful.
